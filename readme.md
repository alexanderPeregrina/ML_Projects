# Machine Learning Projects Repository

This repository contains **hundreds of examples and exercises** covering a wide range of topics in **Python programming** and **Machine Learning**. It is designed as a practical resource for learners and practitioners who want to explore ML concepts through hands-on code and reproducible Jupyter notebooks.

---

## Contents

- **Python Basics**
  - Data types, control structures, functions, OOP
- **Data Handling & Preprocessing**
  - Using `numpy` and `pandas` for numerical and tabular data
- **Visualization**
  - Examples with `matplotlib`, `seaborn`, `plotly`, and other libraries
- **Machine Learning Algorithms**
  - Classification: KNN, Decision Trees, Random Forest
  - Clustering: K-Means
  - Regression models
  - Recommender Systems
  - Support Vector Machines
  - Natural Language Processing
  - Logistic Regression
  - Principal Component Analysis
- **Deep Learning**
  - Multi-Layer Perceptron calssification and regression examples 
  - Examples using `tensorflow` and related frameworks

---

## ‚öôÔ∏è Setting Up the Environment

To ensure reproducibility, a `requirements.txt` file is provided. Follow these steps:

### 1. Create a Virtual Environment
On Windows:
```powershell
python -m venv venv
venv\Scripts\activate
```
On Linux/macOS:
```bash
python3 -m venv venv
source venv/bin/activate
```
### 2. Install Dependencies
Once the environment is active, install all required packages:
```bash
pip install -r requirements.txt
```
##  How to Use
- Clone the repository:
```bash
git clone https://github.com/alexanderPeregrina/ML_Projects.git
cd ML-Projects
```

- Install dependencies from requirements.txt.
